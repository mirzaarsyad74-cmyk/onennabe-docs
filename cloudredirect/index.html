    CloudRedirect Documentation

    🛠️ Step 1: Initial Configuration

    1. Open CloudRedirect.
    2. Go to Mode and click "I understand the risk".
    3. Select CloudRedirect Mode.
    4. Click Setup ➜ Run all patch.

    Troubleshooting

    If you are unable to click the patch button, go to [SUO > Tools > Reinstall Steam], then attempt to run the patch again.

    1. Click Continue and wait for the process to finish.
    2. Click Configure Provider and choose either Google Drive or OneDrive.
    3. Login in the browser window that opens and click Allow.
    4. Click Save Configuration and Restart Steam.

    Cloud Save Integration

    • Steps 1–5 are necessary to pin the manifest.
    • Steps 6–8 are optional and only required if you wish to enable cloud saves.

    🔒 Step 2: How to Lock Game Versions

    Purpose

    Use this feature to prevent Steam from automatically updating your games.

    1. Click on Manifest Pinning.
    2. Select Enable Manifest Pinning.
    3. Find your game name or App ID in the LUA app list.
    4. Enable the toggle to lock that specific version.
    5. Click Restart Steam.
