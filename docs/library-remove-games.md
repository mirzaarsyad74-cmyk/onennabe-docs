# Removing Games from the Library

The Library tab lets you remove one or multiple games at once using the **checkbox selector** and the **🗑️ trash button**.

!!! info "What does removing do?"
    Removing a game deletes its unlock file from your library. This removes the game from Steam's library view. It does **not** uninstall the game files from your PC — your downloaded game data in `steamapps` is kept.

### Step 1 — Select Games Using the Checkbox

Each game in the left sidebar has a **checkbox** (✓) on its left side.

1. Go to the **Library** tab
2. Click the **✓ checkbox** on the left of any game you want to remove
3. The checkbox turns highlighted to show it is selected
4. Repeat for every game you want to remove
5. You can select as many games as you want

!!! tip
    Clicking the game row itself opens the game detail panel. To select a game for removal, click specifically on the **✓ checkbox**, not the game name or icon.

---

### Step 2 — Click the 🗑️ Trash Button

Once at least one game is selected, the **🗑️ trash button** will appear next to the search bar at the top of the library sidebar.

1. Click the **🗑️ trash button**
2. A confirmation dialog will appear showing how many games will be removed
3. Click **OK / Confirm** to proceed
4. The selected games are removed from your library

A success notification will appear confirming how many games were removed.

---

## Summary

```
① Click ✓ checkbox on each game you want to remove
② 🗑️ trash button appears in the search bar area
③ Click 🗑️ → confirm the dialog
④ Games are removed from the library
```

---

## Important Notes

- **Removing does not uninstall.** Your game files remain on disk in `steamapps`. Only the unlock file is deleted.
- **To re-add a removed game**, go back to the Store tab, find the game, and unlock it again.
- **The trash button only appears when at least one game is selected.** If you do not see it, click a checkbox first.
- **To deselect a game**, click its checkbox again — the highlight will be removed and the game will no longer be included in the removal.
